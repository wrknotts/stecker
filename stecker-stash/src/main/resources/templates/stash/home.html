CodeCloud Repository Plugin

This repository plugin for Stecker supports AT&T CodeCloud (Atlassian Stash).

In addition to supporting source files stored in CodeCloud Project Repositories, this plugin is also able to support managed by the Snippets Plugin.

CodeCloud is located at: {{scmEnvironment.baseAddress}}

CodeCloud Repository Files

This plugin supports sharing of source code files that are stored in publicly-accessible Repositories managed by Stash.

To use the bookmarklet for CodeCloud Files:

  1. Add the bookmarklet to your web browser.
  2. Navigate to the file in CodeCloud that you want to share.
  3. Click on the bookmarklet to add new buttons to the CodeCloud UI.
  4. Click on one of these new buttons, to open a dialog box that will allow you to copy the Stecker URL to the clipboard.
  5. Paste the Stecker URL now on the clipboard as appropriate.

CodeCloud Snippets

This plugin supports sharing of source code files that are stored in publicly-accessible Snippets managed by Stash.

To use the bookmarklet for CodeCloud Snippets:

  1. Add the bookmarklet to your web browser.
  2. Navigate to the snippet in CodeCloud that you want to share.
  3. Click on the bookmarklet to add new elements to the CodeCloud UI.
  4. Click on one of these new elements, then copy the Stecker URL to the clipboard.
  5. Paste the Stecker URL now on the clipboard as appropriate.
